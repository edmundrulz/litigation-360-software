\# Litigation 360 Working Rules



\## Runner Windows



Do not type commands.



\### Backend Runner



Purpose:

Run backend only.



\### Frontend Runner



Purpose:

Run frontend only.



\## Workspace Window



All development commands.



\## Testing Window



Only:



\* curl

\* API verification

\* diagnostics



\## Before Any Patch



1\. Backup

2\. Verify

3\. Patch

4\. Syntax Check

5\. Test



\## Never



\* Patch directly in runner windows

\* Delete without quarantine

\* Replace working CRUD logic

\* Skip backup creation
