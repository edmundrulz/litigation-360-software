# Litigation 360 Filename Standard

Format:
YYYY-MM-DD_CLIENT-MATTER_DOCUMENT-TYPE_STAGE_VERSION_SOURCE.ext

Example structure only:
2026-06-18_CLIENTNAME_MATTERNO_AFFIDAVIT_DRAFT_V001_EMAIL.pdf

Rules:
- No unsafe symbols
- No random names
- No final label unless approved
- Use DRAFT, REVIEW, APPROVED, FILED, SERVED
