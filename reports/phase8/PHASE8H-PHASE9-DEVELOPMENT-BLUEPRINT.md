# Phase 8H Phase 9 Development Blueprint

## Purpose
Define the exact Phase 9 build sequence after Phase 8 closeout.

## Phase 9A
Master Client Identity Engine

Build:
- Client duplicate detection
- NRIC/passport/company/email/mobile matching
- Exact/likely/possible/no-match logic
- Duplicate blocking
- Duplicate audit logs

## Phase 9B
Conflict Checking Engine

Build:
- Existing client conflict checks
- Former client checks
- Opposing party checks
- Director/shareholder checks
- Green/Amber/Red conflict rating

## Phase 9C
Matter Numbering Engine

Build:
- MAT-YYYY-DEPT-000001 format
- Sequential numbering
- No duplicate matter numbers
- Department prefixes

## Phase 9D
Matter Intake Wizard

Build:
- Client verification
- Conflict check
- Matter classification
- Document collection
- Fee arrangement
- Lawyer assignment
- Matter activation

## Phase 9E
Workflow Conveyor Engine

Build:
- Civil litigation workflow
- Family workflow
- Corporate workflow
- Auto task creation
- Escalation rules

## Phase 9F
Task Automation Engine

Build:
- Tasks
- Deadlines
- Reminders
- Follow-ups
- Escalations

## Phase 9G
Court Deadline Calculator

Build:
- Filing deadline calculator
- Appeal deadline calculator
- Service deadline calculator
- Weekend/public holiday handling

## Phase 9 Exit Criteria
Phase 9 is complete when Litigation 360 can create a matter through identity verification, conflict review, numbering, workflow selection, task generation, and deadline generation.
