# VALIDATION LIFECYCLE POLICY

## Purpose
This policy governs validation assets across Litigation 360 before Phase 11 promotion.

## Rules
1. Every validation asset must be registered.
2. Every validation asset must have ownership.
3. Every validation asset must map to a domain.
4. Every missing validation must be documented.
5. Validation governance must be reviewed every 30 days.
6. Phase 11 cannot proceed without validation governance PASS.
