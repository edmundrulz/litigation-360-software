# CHANGE GATE VALIDATOR RULES

Required files for every change pack:

01-CHANGE-REQUEST.md
02-IMPACT-ASSESSMENT.md
03-RISK-CONTROL.md
04-TESTING-GATES.md
05-SECURITY-REVIEW.md
06-APPROVAL-WORKFLOW.md
07-DEPLOYMENT-CONTROL.md
08-ROLLBACK-PLAN.md
09-LIVE-MONITORING.md

Blocking Conditions:

- Missing change request
- Missing impact assessment
- Missing testing evidence
- Missing security review
- Missing approval
- Missing rollback plan
- Missing monitoring plan
- Critical unresolved risk
