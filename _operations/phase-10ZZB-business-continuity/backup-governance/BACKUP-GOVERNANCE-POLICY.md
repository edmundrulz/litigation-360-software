# BACKUP GOVERNANCE POLICY

## Backup Frequency

| Data Type | Frequency |
|---|---|
| Source Code | Daily |
| Database | Every 15-30 minutes if possible |
| Documents | Hourly/Daily |
| Reports | Daily |
| Audit Logs | Daily |
| Financial Records | Daily |

## Backup Rule

A backup is not valid until it has been tested.

## Backup Locations

1. Local working copy
2. External drive/NAS
3. Cloud backup
4. Offline archive for critical records

## Verification

Every backup check must record:

- Date
- Person
- Backup source
- Backup destination
- Result
- Issues found
