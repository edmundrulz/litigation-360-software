# CHECKS AND BALANCES REGISTER

## Mandatory Controls

| Control ID | Control | Purpose | Verification |
|---|---|---|---|
| CB-001 | Owner notification | Prevent silent failures | Audit report |
| CB-002 | Lawyer approval | Prevent unauthorized legal action | Approval record |
| CB-003 | Backup verification | Prevent false backup confidence | Recovery test |
| CB-004 | Audit trail | Preserve accountability | Log review |
| CB-005 | Access control | Protect confidential data | Permission review |
| CB-006 | Incident register | Track failures | Incident report |
| CB-007 | File inventory | Confirm structure | Tree report |

## Rule

Every high-risk action must have at least one reviewer and one evidence record.
