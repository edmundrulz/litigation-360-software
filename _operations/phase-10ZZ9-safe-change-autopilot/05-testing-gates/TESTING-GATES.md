# TESTING GATES

No change may be deployed unless these are completed.

## Required Tests

[ ] Build test
[ ] Unit test
[ ] Integration test
[ ] Regression test
[ ] End-to-end test
[ ] Security test
[ ] Database migration test
[ ] Backup and restore test
[ ] User acceptance test
[ ] Smoke test after deployment

## Automatic Failure Conditions

Deployment must stop if:

[ ] Build fails
[ ] Tests fail
[ ] Security scan fails
[ ] Critical bug exists
[ ] Rollback plan missing
[ ] Approval missing
[ ] Documentation missing
[ ] Database backup missing
