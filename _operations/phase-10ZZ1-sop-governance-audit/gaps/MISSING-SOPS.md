## Standard SOP Template
Each missing SOP must include:
1. Purpose
2. Scope
3. Inputs
4. Outputs
5. Parameters
6. Rules
7. Process
8. Validation
9. Operator checklist
10. Escalation path
11. Recovery action
12. Evidence/report path
