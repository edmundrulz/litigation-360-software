## Example: Deadline Page

Current State:
Pending review.

Depends On:
- Sidebar route
- Matter/case data
- Deadline date fields
- Notification rules
- User permissions
- Dashboard alerts
- Reports

Can Break:
- Dashboard accuracy
- User confidence
- Deadline tracking
- Reports
- Notifications

Risk Level:
Medium to High.

Safe Next Action:
Document current behaviour before editing.

Evidence Needed:
- Screenshot
- Route check
- File path
- Current UI behaviour
- Missing function list
