# Master Prompt Library

## Build Prompt

Create this module using:
- exact file paths
- exact commands
- no manual guessing
- test file
- verification report
- rollback instruction

## Audit Prompt

Audit the system and return:
- existing files
- missing files
- broken references
- duplicated files
- required next action

## Handover Prompt

Create a handover package with:
- status
- completed items
- missing items
- risks
- next steps
