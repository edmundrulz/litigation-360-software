# 15 Training Guide

## Purpose
Defines how different users learn and operate Litigation 360.

## User Groups
- Lawyer
- Clerk
- Admin
- Partner
- Developer
- System operator

## Lawyer Training
- View clients
- View cases
- View matters
- Track court dates
- Review documents

## Clerk Training
- Maintain client records
- Update matter information
- Upload and organize documents
- Track deadlines

## Admin Training
- Manage staff
- Monitor health
- Run verification scripts
- Maintain documentation

## Developer Training
- Understand frontend structure
- Understand backend structure
- Run build tests
- Use rollback scripts
- Update documentation after changes
