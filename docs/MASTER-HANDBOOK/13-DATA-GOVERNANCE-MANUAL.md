# 13 Data Governance Manual

## Purpose
Defines how Litigation 360 data should be named, stored, protected, retained, and recovered.

## Core Data Domains
- Clients
- Cases
- Matters
- Court dates
- Documents
- Staff
- Audit logs
- Deployment reports
- Health reports

## Data Rules
- Client data must be accurate.
- Matter data must be linked to the correct client.
- Case data must be linked to the correct matter.
- Documents must be traceable.
- Court dates must be auditable.
- Staff roles must be controlled.
