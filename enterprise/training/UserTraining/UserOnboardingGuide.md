\# Litigation 360 User Onboarding Guide



\## Objective



Enable a new staff member to use Litigation 360 without developer assistance.



\## Day 1



\### Login



\* Access system

\* Authenticate

\* Change password



\### Core Functions



\* Search client

\* Create client

\* Search matter

\* Create matter

\* Upload document

\* Create task

\* Create communication



\### Daily Workflow



1\. Open dashboard

2\. Review tasks

3\. Review deadlines

4\. Review notifications

5\. Update matters

6\. Upload documents

7\. Close completed tasks



\## Competency Target



User can independently operate core legal workflows.
