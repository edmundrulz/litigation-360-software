# CLIENT ONBOARDING PROCESS

## Stage 1 — Lead Qualification
Collect:
- Firm name
- Number of users
- Number of offices
- Practice areas
- Current workflow pain points
- Required modules
- Budget range

## Stage 2 — Demo
Show:
- Dashboard
- Matter flow
- Court flow
- Document flow
- Reports
- Subscription locked features

## Stage 3 — Trial Setup
Create:
- Firm profile
- Trial period
- Admin user
- Feature access
- Training checklist

## Stage 4 — Data Setup
Configure:
- Users
- Roles
- Practice areas
- Matter types
- Templates
- Basic workflows

## Stage 5 — Training
Train:
- Admin
- Lawyer
- Clerk
- Finance
- Partner

## Stage 6 — Go Live
Validate:
- Login
- Role access
- Matter creation
- Document upload
- Reports
- Audit logs
