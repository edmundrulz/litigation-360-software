# SUPPORT SERVICE MODEL

## Support Levels

| Level | Description | Response |
|---|---|---|
| L1 | Basic user issue | Same / next business day |
| L2 | Workflow / configuration issue | Priority support |
| L3 | System / backend issue | Technical escalation |
| L4 | Security / data / outage | Emergency escalation |

## Ticket Categories

- Login
- Billing
- Feature access
- Matter issue
- Document issue
- Report issue
- AI issue
- Data issue
- Performance issue
- Security issue

## Escalation Rule

Any security, data loss, billing lock, or Ground Zero issue must escalate immediately.
