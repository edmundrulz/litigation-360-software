# LICENSING FRAMEWORK

## License Types

1. Single Firm License
2. Multi Office License
3. Enterprise License
4. Government License
5. Ground Zero Founding Client License

## Ground Zero Rule

Father's firm receives permanent unlimited access.

## Standard Customer Rule

All other customers are controlled by:
- Subscription plan
- Payment status
- Trial status
- Role access
- Manual override
- Feature entitlement

## Licensing Matrix

| License | Target | Access | Limitation |
|---|---|---|---|
| Ground Zero | Father's firm | Full | None |
| Single Firm | Small firm | Paid plan | Plan based |
| Multi Office | Multi-branch firm | Advanced | User/office based |
| Enterprise | Large firm | Full commercial | Contract based |
| Government | Gov / agency | Custom | Compliance based |
